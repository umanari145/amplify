# amplify

amplify + vue のサンプル

## 前提インストール
```
npm install -g @vue/cli
npm install -g @aws-amplify/cli
```

## vue インストール
```
npm install aws-amplify
npm install aws-amplify-vue

```


## vueコマンド

### プロジェクト作成
```
vue create プロジェクト名
```

### ホットリロード
```
npm run serve
```

### ビルド
```
npm run build
```

### linter&check
```
npm run lint
```
